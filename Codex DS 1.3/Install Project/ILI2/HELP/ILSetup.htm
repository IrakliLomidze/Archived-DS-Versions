Codex.net 2007 Installation System

Codex.net 2007 is designed for Windows ,XP,2003,2003R2,Vista both for 32bit and 64 bit system.
For Installing Codex.net 2007 you must install pre requisites components on target machine.

When you start installation you may see message "Setup has detected that another program requires the computer to reboot. You must reboot the computer before installing Codex.NET 2007 Prerequisites. Once the system reboots, you need to restart setup. Click OK to reboot. Click Cancel to exit setup and install later. For force installing software check force check box on form and please read readme document..  Force installing has some risks.  Be sure  your system will work fine if you install in force mode ". You can continue installation process but you not have guaranty that all components will install on target machine ( not only codex.net 2007 installation has reboot detector). setup detected another program requires the computer to be rebooted.

How programs  detect that reboot is required and why it is recommended to do restart action.

If application wants to change files witch are  in use, they write  it in following registry key

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\PendingFileRenameOperations
Operation system does operations listed on that registry key before loading those files. So force install is dangerous because you may loose new versions of some files.

If you want to cancel this operation just delete that key (do not empty it, delete it).

 

The following registry key is contains same kind of information

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\RunOnce

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\RunOnceEx
If you want cancel this operation just empty that keys (do not delete it).

 

Pre requisites

  Pre requisites components
  .NET Framework 3.0 Redistributive
  SQL Server 2005 Express With Advances Services
  Microsoft Windows Installer 3.1
  Font Actions

 
.NET Framework 3.0 redistributive

Codex.net 2007 is based on .NET Framework 3.0,   This component is required to be installed.

This module is installing using quiet mode, you can change installation parameters for the installation process by pressing configuration button.

For more information how to configure installation parameters see documentation.

 MSI Installer 3.1 (Microsoft Installer SDK)

For installing codex 2007 software you need to have on target machine MSI 3.1 component. this component is requred alose for .net framework 3.0.

For more information how to configure installation parameters see documentation.

All this component may is already installing on you computer if you use a automatic windows update.

SQL Server 2005 Express Edition with Advances Services SP2 "Codex2007" instance

SQL 2005 Express w/adv with an integrated service pack 2.This is a required component for codex 2007. This component can not e installed by windows update. This module is installing using quiet mode, you can change installation parameters for installation. by press configuration button.

This component installing only in Codex.net 2007 workstation edition. Codex.net 2007 Client and Codex.net 2007 Server is not required this component.

For more information how to configure installation parameters see documentation.

Font Action

Codex.net 2007 using font "Geo ABC" font file name of this font is geoabc.ttf. If in target machine is exists font "GeoABC" (Not "Geo ABC") that is different font with a same file name geoabc.ttf.

For installing Codex.net 2007 Please Remove Font "GEOABC" Manually.

This action is avoid font conflict.
